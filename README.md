## 📜 Resumen 📜

Rediseño de la SuperApp de Yummy. Proyecto especialmente construido para aprender sobre llamados al dom, cambiar los colores de light a dark mode, además de usar revealjs, librería con animaciones de entrada y salida.

## 💻 Si quieres ver el código en tu computadora solo debes hacer esta instalación 💻

- Ubicarse en la carpeta que contendrá el proyecto
- Abrir terminal de comandos
  - git clone https://github.com/rafacodejs/YummyLandingPage.git
  - cd YummyLandingPage
  - Abrir tu editor de código
